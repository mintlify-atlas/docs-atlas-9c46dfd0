---
title: "Delete Recipe"
api: "DELETE /api/v1/recipes/{id}"
description: "Delete a recipe permanently"
---

Permanently deletes a recipe. This action cannot be undone. Only recipes owned by the authenticated user can be deleted.

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the recipe to delete
</ParamField>

## Response

<ResponseField name="ok" type="boolean">
  Confirmation that the recipe was deleted successfully
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE "https://api.fitboard.app/api/v1/recipes/65f3a2b1c4d5e6f7a8b9c0d1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.fitboard.app/api/v1/recipes/65f3a2b1c4d5e6f7a8b9c0d1', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://api.fitboard.app/api/v1/recipes/65f3a2b1c4d5e6f7a8b9c0d1',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)

data = response.json()
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "ok": true
}
```

```json 404 - Not Found
{
  "error": "Not found"
}
```
</ResponseExample>
