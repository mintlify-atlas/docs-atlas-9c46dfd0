---
title: "Update Profile"
api: "PATCH /api/v1/me"
description: "Update the authenticated user's profile information"
---

This endpoint allows the authenticated user to update their profile data. Only the fields provided in the request will be updated.

## Authentication

This endpoint requires authentication. Include a valid JWT token in the Authorization header.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
  
  ```
  Authorization: Bearer <your_token>
  ```
</ParamField>

## Body Parameters

All body parameters are optional. Only include the fields you want to update.

<ParamField body="username" type="string">
  The user's username (3-32 characters)
</ParamField>

<ParamField body="avatar" type="string">
  URL to the user's avatar image (must be a valid URL)
</ParamField>

<ParamField body="active" type="boolean">
  Whether the user account is active
</ParamField>

## Response

Returns the updated user profile with the same structure as the Get Profile endpoint.

<ResponseField name="_id" type="string">
  The user's unique identifier
</ResponseField>

<ResponseField name="email" type="string">
  The user's email address
</ResponseField>

<ResponseField name="emailLower" type="string">
  Normalized lowercase version of the email
</ResponseField>

<ResponseField name="username" type="string">
  The user's updated username
</ResponseField>

<ResponseField name="avatar" type="string">
  URL to the user's updated avatar image
</ResponseField>

<ResponseField name="isAdmin" type="boolean">
  Whether the user has admin privileges
</ResponseField>

<ResponseField name="active" type="boolean">
  Whether the user account is active
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the user account was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the user account was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl -X PATCH https://api.fitboard.com/api/v1/me \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "newusername",
    "avatar": "https://example.com/new-avatar.jpg"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fitboard.com/api/v1/me', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer <your_token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    username: 'newusername',
    avatar: 'https://example.com/new-avatar.jpg'
  })
});

const updatedUser = await response.json();
```
</RequestExample>
