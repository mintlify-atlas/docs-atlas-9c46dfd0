---
title: "List Users"
api: "GET /api/v1/users"
description: "Retrieve a paginated list of all users (admin only)"
---

This endpoint returns a paginated list of all users in the system. This endpoint is restricted to admin users only.

## Authentication

This endpoint requires authentication with admin privileges. Include a valid JWT token in the Authorization header.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication (admin required)
  
  ```
  Authorization: Bearer <your_admin_token>
  ```
</ParamField>

## Query Parameters

<ParamField query="page" type="number" default="1">
  Page number for pagination (minimum: 1)
</ParamField>

<ParamField query="limit" type="number" default="20">
  Number of users per page (minimum: 1, maximum: 100)
</ParamField>

<ParamField query="q" type="string">
  Search query to filter users by email or username
</ParamField>

<ParamField query="active" type="boolean">
  Filter by active status (true/false)
</ParamField>

<ParamField query="sort" type="string" default="-createdAt">
  Sort field (prefix with '-' for descending order)
  
  Examples: `-createdAt`, `username`, `-updatedAt`
</ParamField>

## Response

<ResponseField name="page" type="number">
  Current page number
</ResponseField>

<ResponseField name="limit" type="number">
  Number of users per page
</ResponseField>

<ResponseField name="total" type="number">
  Total number of users matching the query
</ResponseField>

<ResponseField name="users" type="array">
  Array of user objects
  
  <ResponseField name="_id" type="string">
    The user's unique identifier
  </ResponseField>
  
  <ResponseField name="email" type="string">
    The user's email address
  </ResponseField>
  
  <ResponseField name="username" type="string">
    The user's username
  </ResponseField>
  
  <ResponseField name="avatar" type="string">
    URL to the user's avatar image
  </ResponseField>
  
  <ResponseField name="isAdmin" type="boolean">
    Whether the user has admin privileges
  </ResponseField>
  
  <ResponseField name="active" type="boolean">
    Whether the user account is active
  </ResponseField>
  
  <ResponseField name="createdAt" type="string">
    Timestamp when the user account was created
  </ResponseField>
  
  <ResponseField name="updatedAt" type="string">
    Timestamp when the user account was last updated
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.fitboard.com/api/v1/users?page=1&limit=20&active=true" \
  -H "Authorization: Bearer <your_admin_token>"
```

```bash cURL (with search)
curl -X GET "https://api.fitboard.com/api/v1/users?q=john&sort=-createdAt" \
  -H "Authorization: Bearer <your_admin_token>"
```

```javascript JavaScript
const response = await fetch('https://api.fitboard.com/api/v1/users?page=1&limit=20&active=true', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer <your_admin_token>'
  }
});

const data = await response.json();
```
</RequestExample>

## Admin Only

This endpoint is restricted to users with admin privileges. Non-admin users will receive a 403 Forbidden response.
