---
title: "Change Password"
api: "POST /api/v1/change-password"
description: "Change the authenticated user's password"
---

This endpoint allows authenticated users to change their password by providing their current password and a new password.

## Authentication

This endpoint requires authentication. Include a valid JWT token in the Authorization header.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
  
  ```
  Authorization: Bearer <your_token>
  ```
</ParamField>

## Body Parameters

<ParamField body="currentPassword" type="string" required>
  The user's current password for verification
</ParamField>

<ParamField body="newPassword" type="string" required>
  The new password (minimum 8 characters)
</ParamField>

## Response

<ResponseField name="ok" type="boolean">
  Indicates whether the password change was successful
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api.fitboard.com/api/v1/change-password \
  -H "Authorization: Bearer <your_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "currentPassword": "oldPassword123",
    "newPassword": "newSecurePass456"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.fitboard.com/api/v1/change-password', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <your_token>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    currentPassword: 'oldPassword123',
    newPassword: 'newSecurePass456'
  })
});

const result = await response.json();
```
</RequestExample>

<ResponseExample>
```json 200 - Success
{
  "ok": true
}
```

```json 401 - Unauthorized (Invalid Token)
{
  "error": "Unauthorized"
}
```

```json 401 - Invalid Current Password
{
  "error": "Invalid current password"
}
```

```json 404 - User Not Found
{
  "error": "User not found"
}
```
</ResponseExample>

## Password Requirements

The new password must meet the following requirements:
- Minimum 8 characters in length
- Should contain a mix of uppercase, lowercase, numbers, and special characters for security
