---
title: "List Progress Entries"
api: "GET /api/v1/progress"
description: "Retrieve a paginated list of progress entries with optional date range filtering and sorting"
---

Retrieve all progress entries for the authenticated user with pagination support, date range filtering, and flexible sorting options.

## Query Parameters

<ParamField query="page" type="integer" default={1}>
  Page number for pagination (minimum: 1)
</ParamField>

<ParamField query="limit" type="integer" default={20}>
  Number of items per page (minimum: 1, maximum: 100)
</ParamField>

<ParamField query="from" type="string">
  Start date for filtering entries (format: YYYY-MM-DD)
  
  Example: `2025-01-01`
</ParamField>

<ParamField query="to" type="string">
  End date for filtering entries (format: YYYY-MM-DD)
  
  Example: `2025-12-31`
</ParamField>

<ParamField query="sort" type="string" default="-date">
  Sort order for results
  
  Options:
  - `date` - Oldest first
  - `-date` - Newest first (default)
  - `createdAt` - Oldest created first
  - `-createdAt` - Newest created first
</ParamField>
