---
title: "Update Meal"
api: "PATCH /api/v1/meals/{id}"
---

Update a meal with partial or full replacement. Use PATCH for partial updates or PUT for full replacement.

## Partial Update (PATCH)

<api>PATCH /api/v1/meals/{id}</api>

Update specific fields of a meal without affecting other fields.

### Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the meal to update
  ```
  meal_123
  ```
</ParamField>

### Request Body

All fields are optional. Only include the fields you want to update.

<ParamField body="date" type="string">
  Date of the meal in YYYY-MM-DD format
  ```
  2025-11-06
  ```
</ParamField>

<ParamField body="time" type="string">
  Time of the meal in HH:MM format
  ```
  13:00
  ```
</ParamField>

<ParamField body="mealType" type="string">
  Type of meal. One of: `breakfast`, `lunch`, `dinner`, `snack`
  ```
  dinner
  ```
</ParamField>

<ParamField body="notes" type="string">
  Notes or comments about the meal
  ```
  Updated notes
  ```
</ParamField>

<ParamField body="items" type="array">
  Array of food items (replaces entire items array if provided)
  <Expandable title="MealItem Object">
    <ParamField body="name" type="string" required>
      Name of the food item
    </ParamField>
    <ParamField body="amount" type="number" required>
      Quantity of the item
    </ParamField>
    <ParamField body="unit" type="string" required>
      Unit of measurement
    </ParamField>
    <ParamField body="calories" type="number">
      Total calories for this item
    </ParamField>
    <ParamField body="protein" type="number">
      Protein content in grams
    </ParamField>
    <ParamField body="carbs" type="number">
      Carbohydrate content in grams
    </ParamField>
    <ParamField body="fat" type="number">
      Fat content in grams
    </ParamField>
  </Expandable>
</ParamField>

### Response

Returns the updated meal object.

## Full Replace (PUT)

<api>PUT /api/v1/meals/{id}</api>

Completely replace a meal with new data. All required fields must be provided.

### Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the meal to replace
  ```
  meal_123
  ```
</ParamField>

### Request Body

<ParamField body="date" type="string" required>
  Date of the meal in YYYY-MM-DD format
  ```
  2025-11-05
  ```
</ParamField>

<ParamField body="time" type="string">
  Time of the meal in HH:MM format
  ```
  12:45
  ```
</ParamField>

<ParamField body="mealType" type="string">
  Type of meal. One of: `breakfast`, `lunch`, `dinner`, `snack`
  ```
  lunch
  ```
</ParamField>

<ParamField body="notes" type="string">
  Notes or comments about the meal
  ```
  Completely new meal data
  ```
</ParamField>

<ParamField body="items" type="array" required>
  Array of food items consumed in this meal (see MealItem structure in PATCH section)
</ParamField>

### Response

Returns the replaced meal object.

<RequestExample>
```bash cURL (PATCH)
curl -X PATCH "https://api.fitboard.com/api/v1/meals/meal_123" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "notes": "Updated notes",
    "time": "13:00"
  }'
```

```bash cURL (PUT)
curl -X PUT "https://api.fitboard.com/api/v1/meals/meal_123" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "date": "2025-11-05",
    "time": "13:00",
    "mealType": "lunch",
    "notes": "Completely replaced meal",
    "items": [
      {
        "name": "Salad",
        "amount": 200,
        "unit": "g",
        "calories": 150,
        "protein": 5,
        "carbs": 20,
        "fat": 8
      }
    ]
  }'
```

```javascript JavaScript (PATCH)
const response = await fetch('https://api.fitboard.com/api/v1/meals/meal_123', {
  method: 'PATCH',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    notes: 'Updated notes',
    time: '13:00'
  })
});

const updatedMeal = await response.json();
```

```python Python (PATCH)
import requests

response = requests.patch(

        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json={
        'notes': 'Updated notes',
        'time': '13:00'
    }
)

updated_meal = response.json()
```
